# 博客的来源

个人一直想自建一个博客。早期用的 [Hexo](https://hexo.io/zh-cn/)。 然后用的 [Gridea](https://gridea.dev/)，
但给人最大的问题就是没办法完全自己控制输出到样式。在没接触过 [Mdx](https://mdxjs.com/) 之前，自己也尝试过自己用 [markdown-it](https://github.com/markdown-it/markdown-it)
去编译解析 markdown，然后拿到里面的内容自己去构造组件，但是花费了大量的经历在解析上面。直到接触到了 mdx,才发现这一切已经有
人做好了，感谢 mdx 提供的强大能力。
